# 02_Método

## ¿Qué vive en esta carpeta?

En esta carpeta no vive la filosofía de ProgramEze.

Tampoco viven los experimentos concretos ni los aprendizajes obtenidos.

Aquí viven las herramientas que utilizamos para investigar la realidad.

Nuestro objetivo no es construir un método rígido.

Nuestro objetivo es construir instrumentos que nos ayuden a observar mejor los fenómenos, reducir nuestros propios sesgos y aprender de manera honesta.

El método no existe para reemplazar el criterio humano.

Existe para protegerlo.

---

## Qué esperamos encontrar aquí

Con el tiempo esta carpeta contendrá distintos tipos de herramientas:

- Instrumentos para observar fenómenos.
- Guías para registrar evidencia.
- Manuales para formar nuevos observadores.
- Procedimientos para reducir sesgos.
- Marcos metodológicos que hayan demostrado utilidad en la práctica.

Nada de esto se escribe por anticipado.

Cada documento debe nacer porque la realidad lo hizo necesario.

---

## Una regla importante

En ProgramEze intentamos resistir una tentación constante:

> Escribir documentación para problemas que todavía no existen.

Preferimos una carpeta vacía antes que un documento construido únicamente desde la imaginación.

Cuando un nuevo documento aparece aquí, idealmente lo hace porque algún experimento nos obligó a escribirlo.

La realidad siempre tiene prioridad sobre nuestras teorías.
