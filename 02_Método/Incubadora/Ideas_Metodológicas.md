# Idea 001 — El Modo Auditor no parece activarse automáticamente

## Observaciones relacionadas

- Experimento_001
- Experimento_002

## Descripción

Hasta el momento parece que una lectura superficial del material no activa el criterio profundo del docente.

El criterio comienza a aparecer cuando el observador cambia del modo "leer" al modo "auditar".

Todavía desconocemos si este cambio ocurre por:

- una segunda lectura;
- un cambio deliberado de actitud;
- mayor tiempo de contemplación;
- o una combinación de estos factores.

## Qué necesitamos observar

- ¿Siempre ocurre?
- ¿Ocurre en otros docentes?
- ¿Ocurre en otras materias?
- ¿Puede activarse desde la primera lectura?

# Idea 002 — El prerregistro podría reducir el sesgo

## Descripción

Antes de comenzar un experimento podría registrarse explícitamente qué esperamos encontrar.

Sin embargo, el prerregistro no debería indicar lugares específicos donde esperamos fricción, porque eso podría contaminar la observación.

## Qué necesitamos observar

- ¿El prerregistro mejora la falsabilidad?
- ¿El prerregistro puede producir sesgo de confirmación?
- ¿Conviene registrar hipótesis generales en lugar de predicciones por diapositiva?

# Idea 003 — Toda hipótesis necesita su propio intento de ruptura

## Descripción

Cada vez que una hipótesis comience a fortalecerse, deberíamos diseñar deliberadamente un experimento cuyo objetivo sea intentar demostrar que esa hipótesis es incorrecta.

No buscamos proteger nuestras ideas.

Buscamos descubrir hasta dónde resisten.

---

# Idea 004 — El instrumento también evoluciona

## Descripción

Los experimentos no solo modifican nuestras hipótesis.

También modifican al observador.

Cada nuevo experimento podría volver más sensible nuestro propio criterio.

Por lo tanto, una mayor cantidad de observaciones no implica necesariamente una mayor cantidad de problemas.

Puede significar simplemente que el instrumento de observación mejoró.

---

# Idea 005 — Separar claramente fenómeno de herramienta

## Descripción

Cuando observamos un problema debemos distinguir cuidadosamente entre:

- un problema del fenómeno observado;
- un problema de la herramienta utilizada para observarlo.

Ejemplo:

Un error de renderizado no constituye evidencia pedagógica.

Un error de maquetación no implica necesariamente un error conceptual.

# Idea 006 — Diferenciar decisión tomada de impulso no ejecutado

## Descripción

Durante una observación puede existir una diferencia importante entre:

- una decisión efectivamente tomada;
- una decisión que el observador quiso tomar pero dejó en suspenso.

Todavía no sabemos si esta distinción será necesaria en la plantilla.

Pendiente de validación durante el uso real del instrumento.

## Descripción

Las categorías "Duda leve" e "Intuición débil" podrían superponerse.

Todavía no sabemos si el observador las usará de forma distinguible.

# Idea 008 — El preregistro podría requerir evidencia temporal independiente

## Descripción

Si el preregistro se completa dentro del mismo documento que las observaciones, podría modificarse después de observar el material.

Una posibilidad futura sería registrar el preregistro en un archivo separado o commit previo.
